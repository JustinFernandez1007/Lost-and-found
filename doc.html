Documentation

This page provides instructions and records for the School Lost & Found system.

How to Use

  • Report found items using the Report page.
  • Browse items using the Browse page.
  • Claim items by clicking Claim on an approved item.
  • Admins can approve, mark claimed, archive items, and manage claims in the Admin Panel.

Roles

  • Viewer: Can browse approved items.
  • Staff: Can report items and browse.
  • Admin: Full permissions to approve, claim, archive, and manage claims.

Printable Records

You can use your browser's print function (Ctrl+P or Cmd+P) to print this documentation page for physical records.

Technical Notes

  • Backend: Flask with SQLite database.
  • Static files: CSS in static/style.css, images in static/uploads/.
  • Templates: Jinja2 in templates/ folder.
  • Roles and permissions are enforced in app.py.
